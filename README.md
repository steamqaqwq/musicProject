# MusicProject

**音乐网站**:musical_note:

- 前端：Vue2+Vuex+ElementUi+Vue Router+Echarts
- 后端 ：java+SpringBoot

描述：完成音乐网站基本功能，如音乐播放器，搜索，排行榜，专辑，点赞，收藏等。并通过后台管理系统对用户，歌曲，歌手等进行基本管理。
